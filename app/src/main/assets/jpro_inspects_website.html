J-Pro Inspects - Website

It's a Home Inspection business website, which provides comprehensive information about their business, terms and conditions, credibility, references and qualifications.

This responsive website uses jQuery, imports an external Google font, makes heavy use of CSS selectors.

I've either designed the art or customized the jQuery UI theme; Patrice Paulino, my wife, has taken all other professional pictures.

It was fun to design with mobile integration in mind, since the Android App is currently using the custom web service to exchange data with the website.
