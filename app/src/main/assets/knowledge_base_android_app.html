knowledge Base - Android App

An Android App that allows users to create companies in which the administrators enroll problems/solutions and employees can search and view them.

Depending on the login, data is downloaded from a custom web-service into the device's local SQLite tables.

This App offers language selection of four languages: English, Spanish, French or Portuguese.

Once the user chooses one, it gets stored in an SQLite local database, so the user does not have to select it again, unless another language selection is needed.
