ECLIPSE

Great IDE

I've adopted Eclipse for a number of languages as I write code, including Java, PHP, JavaScript among others.

The predictive text is great, scope recognition and very reliable.

Plugins

Before Android Studio came out, I used to develop my Android apps through Eclipse, by installing the appropriate plugins.

It was a great experience, however Google strongly advised that Eclipse Android developers should switch to Android Studio, so I did, but I know Eclipse is a great tool.
