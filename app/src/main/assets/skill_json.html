JSON

Handy

Through the last 10 years or so, I've been using JSON for data manipulation in a number of enterprise applications.

JSON is a language-independent data format. It derives from JavaScript and code to generate and parse JSON-format data is available in many programming languages.

JSON Schema is based on the concepts from XML Schema (XSD), but is JSON-based. The JSON data schema can be used to validate JSON data. As in XSD, the same serialization/deserialization tools can be used both for the schema and data. The schema is self-describing.
