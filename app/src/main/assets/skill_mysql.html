MYSQL

Good Engine

I've been deploying applications that rely on MySQL servers for 1999.

It has served me quite well from all kinds of platforms, as it is still my public website's and personal application's main database engine.

Along with MySQL Workbench, I get to manage any database in a matter of minutes. It's a great combo.

I have to give credit to MySQL because throughout my carreer, it surely has educated me about database design, management, data flow, keys, triggers and connectivity.
