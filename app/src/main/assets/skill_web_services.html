WEB SERVICES

Essential modernization

Early in my carreer I realized how necessary it would be to have a process that works for various applications at once.

I came across SOAP and immediately realized the virtue of having a web service at my disposal.

Soon after, I learned that I could design my own web service, so I could better handle my web requests across platforms.

When integrated with the right data, most likely in JSON or XML, web services are the way of the future.

They are pretty universal these days, but extremelly efficient.

I love writing my own web services, especially when deploying mobile and web applications.

Productivity is great!
