WINDOWS

First Class OS

Since 1995, I've been working with MS-DOS, then Microsoft Windows.

All it's distributions had such unique features, but the more recent ones have are solid and very reliable.

From account/power/network management to command prompt usage, I've experienced almost every aspect of the operating systems.

Throughout my MS-DOS/Windows development, I have even setup quite a few different servers, such as Apache HTTP/PHP, MySQL, IIS, SQL, DB2, FTP, SMTP, POP3, File Server, Media Server... you get the point!

Microsoft Windows is an elegant operating system.
